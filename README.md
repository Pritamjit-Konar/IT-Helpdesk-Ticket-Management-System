## Project Overview
This project is a simulated IT Helpdesk Ticket Management System created using Google Forms and Google Sheets.  
It demonstrates practical understanding of IT support workflow, ticket handling process, and basic reporting.

## Objective
To showcase real-world understanding of:
- IT Support Operations  
- Ticket Tracking Process  
- Basic Data Management  
- Reporting and Analysis  

## Tools Used
- Google Forms  
- Google Sheets  
- Data Validation  
- Conditional Formatting  
- Pivot Tables  
- Charts  

## Features Implemented
- User-friendly ticket submission form  
- Automated data collection  
- Issue categorization  
- Priority-based tracking  
- Ticket status management  
- Assignment to support team  
- Dashboard for reporting  

## Workflow
1. User submits issue through Google Form  
2. Data stored in Google Sheets  
3. Support team updates ticket status  
4. Dashboard reflects progress
